## Features/highlights

## Setup
```bash
$ git clone https://github.com/Williams94/art-shop.git
$ cd shop
$ npm i
$ npm start
```

## Build
```bash
$ npm run build
```

## Test the build
To test prpl-server build:
```bash
$ npm run serve:prpl-server
```
To test static build:
```bash
$ npm run serve:static
```
